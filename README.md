# Appfigurate™ ObjCExample

**This repository is now archived (at version 4.0.0), no further updates are planned to ObjCExample. Appfigurate Library 5.0.0+ continues to support integration with Objective-C (for existing projects). Please view [SwiftExample](https://github.com/electricbolt/appfiguratesdk-swiftexample) going forward.**

Demonstrates how to integrate Appfigurate SDK into an Objective-C, iOS and watchOS, app and app extension.

Documentation for Appfigurate is available at [electricbolt.gitbook.io](https://electricbolt.gitbook.io).
